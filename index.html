1. Carpet Kit Layout (CCS0361 Example)

Install in numbered order (sills first, front mats last). Dashed #10 overlays.

#NameQtyNotes

2Front Bulkhead1Steering cutout

10Front Floor Mats2Overlay last

2. Tools & Materials

  • Contact cement (Evostik, Weldwood) for sills; spray (3M 90, Tetrosyl) for light hold.
  • Knife/shears; roller/block; tape measure; heat gun.
  • Clips (spiky prong + silver washer); pop-rivet studs.
  • Cleaner (alcohol); masking tape.
